# OOP

- 우리가 보는 모든 물건은 object로 본다
- 이를 주어 동사로 표현한다 => 철학자들의 관점
- 분류화 계층화 => 모든 세상을 표현할 수 있다
- 이런 개념으로 class가 나왔어

---

# Class

- class 는 분류다
- object 특성들을 묶어서 관리하는데 목적
- 이는 틀이야 붕어빵 틀
- 이 틀로 찍어낸 것이 instance 붕어빵이다
- 이 붕어빵 자체를 object = instance
- class를 언제든지 불어내어서 조작가능하다(기능추가) =>루비에서만 가능한 소오르음~

# Module
